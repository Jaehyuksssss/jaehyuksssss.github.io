# 피그마 뽀이 게임 기획서

## 1. 게임 컨셉

### 게임명
**"피그마 뽀이 - 컬러 매칭"** 또는 **"뽀이의 색깔 퍼즐"**

### 컨셉
피그마(Figma)의 마스코트인 뽀이를 주인공으로 한 색상 매칭/조합 퍼즐 게임. 디자인 툴의 특성을 살려 색상, 레이어, 컴포넌트 등의 개념을 게임화.

---

## 2. 게임 플레이 방식

### 핵심 메커니즘
**"목표 색상을 만들기 위해 주어진 색상들을 조합하거나 선택하는 퍼즐 게임"**

### 게임 모드 제안

#### 모드 A: 색상 조합 퍼즐 (추천)
- **플레이 방식**: 
  - 화면 상단에 목표 색상(예: `#FF6B6B`)이 표시됨
  - 하단에 여러 색상 옵션들이 주어짐 (예: 빨강, 파랑, 노랑 등)
  - 플레이어는 주어진 색상들을 조합하거나 직접 선택하여 목표 색상에 가장 가까운 색을 만들어야 함
  - RGB/HSL 값을 조정하는 방식도 가능

- **점수 시스템**:
  - 목표 색상과의 차이(Delta E)를 계산하여 정확도 점수 부여
  - Perfect (차이 < 5): 100점
  - Great (차이 < 15): 80점
  - Good (차이 < 30): 50점
  - Miss: 0점

- **난이도 진행**:
  - 초반: 명확히 구분되는 색상 (빨강, 파랑, 노랑 등)
  - 중반: 비슷한 색상 구분 (다양한 빨강 계열)
  - 후반: 미묘한 차이의 색상 (베이지, 크림, 아이보리 등)

#### 모드 B: 레이어 스택 퍼즐
- **플레이 방식**:
  - 피그마의 레이어 개념을 활용
  - 여러 레이어(색상 블록)를 쌓아서 최종 색상을 만듦
  - 레이어의 순서와 투명도(blend mode)가 결과에 영향
  - 목표 색상에 맞추기

#### 모드 C: 컴포넌트 찾기
- **플레이 방식**:
  - 여러 컴포넌트(뽀이 캐릭터) 중에서 특정 색상 조합을 가진 것을 찾기
  - 시간 제한 내에 정확하게 찾기

## 3. 게임 구조

### 화면 구성
```
┌─────────────────────────────┐
│  목표 색상: [색상 박스]     │
│  RGB: (255, 107, 107)       │
├─────────────────────────────┤
│                             │
│      뽀이 캐릭터 영역        │
│      (애니메이션)           │
│                             │
├─────────────────────────────┤
│  선택 가능한 색상 팔레트    │
│  [빨강] [파랑] [노랑] ...   │
│                             │
│  [조합하기] [제출]          │
└─────────────────────────────┘
```

### 게임 상태
- `idle`: 게임 시작 전
- `playing`: 플레이 중
- `result`: 결과 표시
- `levelup`: 레벨 업 화면

## 4. 기술적 구현 방향

### 색상 계산
- **Delta E 계산**: 목표 색상과 플레이어가 만든 색상의 차이를 정확히 측정
- **RGB/HSL 변환**: 색상 조합 시 RGB 또는 HSL 값으로 계산
- **색상 조합 로직**: 
  - 두 색상을 섞을 때: 평균값 또는 가중 평균
  - 여러 색상 조합: RGB 각 채널의 평균

### 애니메이션
- 뽀이 캐릭터가 색상을 맞췄을 때 기쁨 표현
- 틀렸을 때 실망 표현
- 레벨 업 시 축하 애니메이션

### 난이도 조절
- 레벨에 따라 목표 색상의 복잡도 증가
- 색상 옵션의 수 증가 (3개 → 5개 → 7개)
- 시간 제한 추가 (선택적)

- 레벨/스테이지 표시

## 6. 점수 및 진행 시스템

### 점수 계산
```
기본 점수 = 100 - (Delta E / 최대 차이 * 100)
보너스 = 시간 보너스 (빠를수록 높음)
최종 점수 = 기본 점수 + 보너스
```

### 레벨 시스템
- 레벨 1-5: 기본 색상 (빨강, 파랑, 노랑, 초록, 보라)
- 레벨 6-10: 중간 색상 (주황, 분홍, 청록 등)
- 레벨 11+: 고급 색상 (베이지, 크림, 네이비 등)

### 기록 시스템
- 최고 점수 저장 (localStorage)
- 레벨별 클리어 기록
- 평균 정확도 기록

## 7. 게임 흐름

### 시작 화면
1. 게임 제목 및 뽀이 캐릭터 표시
2. "게임 시작" 버튼
3. 난이도 선택 (쉬움/보통/어려움)

### 게임 플레이
1. 목표 색상 표시 (3초)
2. 색상 팔레트에서 선택/조합
3. "제출" 버튼 클릭
4. 결과 표시 (정확도, 점수)
5. 다음 라운드 또는 레벨 업

### 종료 화면
1. 최종 점수 표시
2. 최고 기록 비교
3. "다시하기" / "메인으로" 버튼

## 10. 참고사항

### 기존 게임들과의 차별점
- **ReactionGame**: 반응속도 중심 → **색상 인지 및 조합 능력** 중심
- **TimeMatch**: 시간 감각 → **색상 감각**
- **TraceRunner**: 기억력 → **색상 조합 능력**

### 타겟 유저
- 디자이너 또는 디자인에 관심 있는 사용자
- 색상 감각을 테스트하고 싶은 사용자
- 간단한 퍼즐 게임을 즐기는 사용자
